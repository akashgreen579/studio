# **App Name**: TestCraft AI Project Manager

## Core Features:

- Role-Based Landing: Detect user role on login to display appropriate dashboard elements and CTAs (Create Project for managers, Your Projects for employees).
- Project Creation Wizard: A modal wizard to guide managers through project creation, including name, description, team member assignment, and permission presets.
- Team Member Assignment: Search and select multiple team members during project creation.
- Permission Presets: Apply permission presets (Manager, Senior QA, Tester) or customize individual permissions.
- Project Summary Card: Display key project details (name, owner, members, last updated) and quick links after project creation.
- Employee Project List: Show assigned projects to employees with corresponding access badges.
- Audit Log: Log all create, assign, and permission-change actions with user and timestamp details.

## Style Guidelines:

- Primary color: Deep Blue (#293B5F) for a professional and trustworthy feel.
- Background color: Light Gray (#E8E8EA) to provide a clean and calm workspace.
- Accent color: Soft Violet (#845EC2) for CTAs and important highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif, providing a modern and neutral look for readability and professionalism.
- Use clear and consistent icons to represent actions, permissions, and project status.
- Employ a clean and structured layout with clear visual hierarchy to ensure ease of use and navigation.
- Incorporate subtle transitions and feedback animations to enhance the user experience and provide confirmation on actions.
